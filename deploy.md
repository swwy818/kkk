# 📱 手机运行指南

## 🎯 最简单的方法：GitHub Pages

### 第一步：上传到GitHub
1. 登录 [GitHub.com](https://github.com)
2. 点击右上角 "+" → "New repository"
3. 仓库名输入：`wechat-simulator`
4. 选择 "Public"
5. 勾选 "Add a README file"
6. 点击 "Create repository"

### 第二步：上传文件
将以下4个文件上传到仓库：
- `index.html`
- `styles.css`
- `script.js`
- `README.md`

### 第三步：启用GitHub Pages
1. 进入仓库页面
2. 点击 "Settings" 选项卡
3. 在左侧菜单找到 "Pages"
4. Source 选择 "Deploy from a branch"
5. Branch 选择 "main"
6. 点击 "Save"

### 第四步：获取网址
- 等待2-3分钟
- 网址格式：`https://你的用户名.github.io/wechat-simulator`
- 用手机浏览器打开即可使用

## 📱 手机使用技巧

### 添加到主屏幕
**iPhone Safari：**
1. 打开网站
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

**Android Chrome：**
1. 打开网站
2. 点击右上角菜单
3. 选择"添加到主屏幕"
4. 点击"添加"

### 最佳体验
- 建议使用Chrome或Safari浏览器
- 可以全屏使用，体验接近原生App
- 支持离线使用（数据本地存储）
